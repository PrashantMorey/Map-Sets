"Q:1 You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string.

Input:

abcadeecfb

Output:

abcdef"

      Ans:
      let str="abcadeecfb";
      const s = new Set();
      for(let i=0;i < str.length;i++){
          s.add(str[i]);
      }
      console.log(...s);

"Q:2 You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)

Input:

abcadeecfb

Output:

a=2

b=2

c=2

d=1

e=2

f=1"

    Ans:
    let word="abcadeecfb";
    let map = new Map();
    for(let i=0;i < word.length;i++){
        if(map.has(word[i])){
            map.set(word[i],Number(map.get(word[i]))+1);
        }
        else{
            map.set(word[i],1);
        }
    }
    for(const [key,value]of map){
        console.log(key +" = " + value);
    }
